Подробное описание

The QsciLexerPascal class encapsulates the Scintilla Pascal lexer.

Перечисления

anonymous enum

This enum defines the meanings of the different styles used by the C++ lexer.

Элементы перечислений
Default 

The default.

Identifier 

An identifier.

Comment 

A '{ ... }' style comment.

CommentParenthesis 

A '(* ... *)' style comment.

CommentLine 

A comment line.

PreProcessor 

A '{$ ... }' style pre-processor block.

PreProcessorParenthesis 

A '(*$ ... *)' style pre-processor block.

Number 

A number.

HexNumber 

A hexadecimal number.

Keyword 

A keyword.

SingleQuotedString 

A single-quoted string.

UnclosedString 

The end of a line where a string is not closed.

Character 

A character.

Operator 

An operator.

Asm 

Inline Asm.
