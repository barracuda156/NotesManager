Подробное описание

To optimise processing of document modifications by DocWatchers, a hint is passed indicating the scope of the change. If the DocWatcher is a document view then this can be used to optimise screen updating.

Данные класса

int DocModification::linesAdded

Negative if lines deleted.

const char* DocModification::text

Only valid for changes to text, not for changes to style.
