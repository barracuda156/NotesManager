Подробное описание

The QsciLexerVHDL class encapsulates the Scintilla VHDL lexer.

Перечисления

anonymous enum

This enum defines the meanings of the different styles used by the VHDL lexer.

Элементы перечислений
Default 

The default.

Comment 

A comment.

CommentLine 

A comment line.

Number 

A number.

String 

A string.

Operator 

An operator.

Identifier 

An identifier.

UnclosedString 

The end of a line where a string is not closed.

Keyword 

A keyword.

StandardOperator 

A standard operator.

Attribute 

An attribute.

StandardFunction 

A standard function.

StandardPackage 

A standard package.

StandardType 

A standard type.

KeywordSet7 

A keyword defined in keyword set number 7. The class must be sub-classed and re-implement keywords() to make use of this style.
