Функция создает действие и возвращает указатель на него.

Происходит создание действия и установка его свойств, переданных в параметрах. Вот так будет создана кнопка:

QAction * action = new QAction( this );
action->setText( "Открыть" );
action->setToolTip( "Открытие ..." );
action->setWhatsThis( "Действие открывает ..." );
action->setShortcut( QKeySequence( QKeySequence::Open ) );
action->setIcon( QIcon( ":/icon" ) );
action->setCheckable( true );

Тоже самое, но с использование данной функции:

QAction * action = Create::Action::get( this, "Открыть", "Открытие ...", "Действие открывает ...", QIcon( ":/icon" ), QKeySequence( QKeySequence::Open ), true );
Аргументы
textназвание действия
toolTipтекст всплывающей подсказки
whatsThisтекст подсказки What's This действия
iconиконка на действие
keyкомбинация "горячей клавиши"
checkableданное свойство указывает, является-ли действие контрольным
Возвращает
указатель на созданное в функции действие
